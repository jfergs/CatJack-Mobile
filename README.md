# CatJack Mobile

Native iOS and iPadOS amateur radio client.

## Vision
Direct USB radio control from iPhone/iPad with CAT control, APRS, logging, POTA/SOTA support, audio transport, and future JS8/FT8 digital modes.

## Milestones
- v0.1 Radio Control
- v0.2 APRS
- v0.3 Logging & POTA
- v0.4 Audio Foundation
- v1.0 JS8 Support
